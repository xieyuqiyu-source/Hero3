# Hero3 GM 后台

`Hero3 GM 后台` 是 Hero3 的管理端前端工程，当前基于 `React + TypeScript + Vite` 搭建。第一版只提供静态后台骨架和 mock 数据，用于提前确定运营工具的信息架构。

## 当前范围

- 运营总览指标
- 后端健康状态读取
- demo 玩家状态只读预览
- 资源调整入口禁用占位
- 系统操作入口
- 操作审计日志
- 高危操作规范提示

当前只读取 `/healthz` 和 `/api/v1/game/state`，不会执行实际 GM 操作。后续需要接入管理员登录、权限校验、审计写入和 Go 后端 GM API。

## 本地开发

安装依赖：

```bash
pnpm install
```

启动开发服务器：

```bash
pnpm dev
```

默认地址：

```text
http://localhost:5174
```

构建：

```bash
pnpm build
```

代码检查：

```bash
pnpm lint
```

## 开发约定

- GM 后台是高风险工具，高危操作必须二次确认。
- 资源调整、封禁、配置刷新等操作必须进入审计日志。
- 在管理员身份、权限和审计链路完成前，页面只允许展示 mock 数据。
- 页面需要兼容窄屏展示，但后台优先保障桌面端信息密度。
